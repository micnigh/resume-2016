# Requirements/Recommended

 - [node] `v4+`
 - [atom]
    - Packages
      - [atom-typescript]
      - [atom-lint]
      - [linter-tslint]
 - [docker]

# Quick start

```bash
# first time
npm install -g gulp typings bower
npm install

# every time
gulp --gulpfile gulpfile.transpile.js watch
gulp watch
```

# Features

 - [typescript] - javascript type support
 - [gulp] - automated build system
